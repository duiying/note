# Ansible简介与安装

> Ansible介绍  

- 概念: Ansible是用Python语言开发的一个开源部署工具
- 特点: 使用SSH协议通讯, 全平台, 无需编译, 模块化部署管理
- 作用: 推送Playbook进行远程节点快速部署

> Ansible优势  

- 轻量级无客户端(Agentless)
- 开源免费, 学习成本低, 快速上手
- 使用playbook作为核心配置架构, 统一的脚本格式, 批量化部署
- 支持目前主流的开发场景
